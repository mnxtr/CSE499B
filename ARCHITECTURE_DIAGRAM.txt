                              KEY FEATURES
================================================================================

🤖 LLM Chatbot:
   • Knowledge-based (no external API)
   • Context-aware responses
   • Traffic sign information
   • Natural language understanding
   • Conversation history

📊 Analytics:
   • Real-time event tracking
   • Session management
   • Performance metrics
   • User interaction logging
   • Chat history
   • Privacy-focused (local only)

🌐 Web Interface:
   • Multiple input modes
   • Model selection
   • Live chatbot
   • Analytics dashboard
   • Responsive design

🔒 Privacy:
   • All data local (SQLite)
   • No external services
   • No personal info collected
   • Random session IDs

================================================================================
                           TECHNOLOGY STACK
================================================================================

Frontend:       Gradio 6.0.1 (Python-based web UI)
Detection:      YOLOv11 (Ultralytics)
Analytics:      SQLite3 (built-in Python)
Chatbot:        Knowledge-based (no external LLM)
Backend:        Python 3.12
Dependencies:   OpenCV, NumPy, PyTorch
